## GoogleApi v3.0.0
Simple and easy Google Api integrations.  
Supports Maps, Places, Roads, Custom Search and Translate.  
https://www.nuget.org/packages/GoogleApi (net4.5, netstandard1.2)


NOTE: Feel free to report any issues, as well as create a PR for suggested changes / fixes.  


### Google Api Integrations:
##### Google Maps
  * Directions
  * Distance Matrix
  * Elevation
  * Geocoding 
  * Reverse Geocoding
  * Geolocation
  * Roads 
    * Nearest Roads
    * Snap To Roads
    * Speed Limits
  * Time Zone

##### Google Places
  * Place Search
    * Near By
	* Text
	* Radar
  * Place Details
  * Place Add
  * Place Delete
  * Place Photos
  * Place Autocomplete
  * Query Autocomplete

##### Google Search (*Alpha*)
  * Web
  * Image
  * Enterprise (Paid)

##### Google Translate 
  * Detect (Done)
  * Languages (Done)
  * Translate (Done)

##### Google Functions 
  * MergePolyLine
  * EncodePolyLine
  * DecodePolyLine
